## Overview

Clone configured instances instantly. No repetitive setup, 75% faster.

## Quick Start

```javascript
// 1. Create and configure base instance
const base = await client.instances.create({
  name: 'calculator-base',
  description: 'Python calculator with deps'
});

await base.run('Setup calculator.py with numpy and pandas');

// 2. Clone instantly
const clone = await client.instances.createFromTemplate({
  templateName: 'calculator-base',
  newInstanceName: 'calc-clone-1'
});

// 3. Use immediately (inherits all files & deps)
await clone.run('Calculate average from data.csv');
```

## What's Cloned

✅ **Inherited**: description, env, region, vmSize, all files, installed packages  
❌ **Not inherited**: name (you provide new name)

## vs Container Templates

| Feature | Instance Clone | Container Template |
|---------|---------------|-------------------|
| Method | `createFromTemplate()` | `templates.create()` |
| Speed | Instant | 2-5 minutes |
| Clones | Files & deps | Docker image |
| Use case | Reuse workspace | Custom OS/packages |

## Save 75% on Setup

```javascript
// ❌ Setup every time
for (let i = 0; i < 10; i++) {
  const agent = await client.instances.create({
    name: `agent-${i}`,
    description: 'Agent'
  });
  await agent.run('Install packages'); // 5 min each
  await agent.run('Do work'); // 2 min
}
// Total: 70 minutes

// ✅ Setup once, clone many
const base = await client.instances.create({
  name: 'base',
  description: 'Base agent'
});
await base.run('Install packages'); // 5 min once

for (let i = 0; i < 10; i++) {
  const clone = await client.instances.createFromTemplate({
    templateName: 'base',
    newInstanceName: `agent-${i}`
  });
  await clone.run('Do work'); // 2 min
}
// Total: 25 minutes (75% faster!)
```
